Traffic Light System – Arduino‑Based Intersection and Railway Crossing
Built a functional 4‑way traffic‑light intersection paired with an automated railway‑crossing mechanism using Arduino (C++). The model was constructed on a cardboard base with measured road layouts, and the traffic lights were assembled from popsicle sticks, straws, LEDs, and soldered wiring routed through the poles and concealed beneath the surface. The system was programmed with a timed 15‑second light cycle to simulate realistic intersection behavior.
A railway‑crossing barrier was added to emulate scheduled train movement. The gate was driven by a servo motor and programmed to lower every two minutes according to a hypothetical train timetable, pausing road traffic and coordinating with the light sequence. This created a cohesive and realistic multi‑system traffic model.
Technologies Used:
Arduino (C++), breadboard wiring, soldering, LEDs, servo motors, timing logic
Key Features:
• Realistic traffic‑light sequencing
• Automated railway‑crossing gate synchronized with train intervals
• Clean physical prototype with concealed wiring and structured layout
